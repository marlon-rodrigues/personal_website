Traffic Sign Classifier

Python, TensorFlow, OpenCV

In this project, I used what I've learned about deep neural networks and convolutional neural networks to classify traffic signs. I trained and validated a model using TensorFlow so it could classify traffic sign images using the German Traffic Sign Dataset. After the model was trained, I tried out the model on images of German traffic signs that I found on the web. My model final results gave me an outstanding accuracy of 99.9% on the training set, 98.5% on the validation set and a solid 96.4% on the testing data set. I pushed it further and tried on 13 different german traffic signs I found on line and the model had an accuracy of 84.6% (11 of the 13 images were predicted correctly), which compares favorably to the accuracy on the test set of 96.4%. See full write up report.

Behavioral Cloning

Python, Keras, OpenCV

In this project, I used what I've learned about deep neural networks and convolutional neural networks to clone driving behavior. I built and trained a convolutional neural network for end-to-end driving in a simulator, using TensorFlow and Keras. I also used optimization techniques such as regularization and dropout to generalize the network for driving on multiple tracks. I used a simulator provided by Udacity where I could steer a car around a track for data collection. I used image data and steering angles to train a neural network and then used this model - which outpus a steering angle - to drive the car autonomously around the track. See full write up report.

Extended Kalman Filter

C++, Motion Models, Lidar and Radar Sensor Fusion

In this project I implemented in C++ a kalman filter to estimate the state of a moving object of interest with noisy lidar and radar measurements. The video shows the results of the kalman filter predictions. Lidar measurements are red circles, radar measurements are blue circles with an arrow pointing in the direction of the observed angle, and estimation markers are green triangles. The video shows what the simulator looks like when a c++ script is using its Kalman filter to track the object. The simulator provides the script the measured data (either lidar or radar), and the script feeds back the measured estimation marker, and RMSE values from its Kalman filter. See source code.
